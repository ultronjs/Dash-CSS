Responsive Image

Images are used to represent something. Dash CSS has Responsive Images. It will shrink and grow according to Container Size.

Example of Responsive Images

Below is the example of Responsive Image.To make a image Responsive in your project user just need to add img_responsive class to its img tag.That will make the image Responsivein nature.
