Avatars

Avatars are images that users can set as their profile picture. On GitHub, they're always going to be rounded squares. They can be custom photos, uploaded by users, or generated as Identicons as a placeholder.

Basic Example

Add .avatar to any img element to make it an avatar.

Small Avatar

We occasionally use smaller avatars. Anything less than 24px wide should include the .avatar-small modifier class to reset the border-radius to a more appropriate level.

Avatar Sizes

Instead of using the width and height attribute, you can also use a class like .avatar-[1-8]. The sizes go from 16px up to 64px
